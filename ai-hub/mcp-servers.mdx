**MCP** significa **Model Context Protocol**, una función del AI Hub que simplifica conectar la IA a herramientas externas — **sin necesidad de configurar endpoints de API complejos**.

<Card type="note">
  Video introductorio: [youtu.be/qHVao5KHzx0](https://youtu.be/qHVao5KHzx0?si=Pw4VHO2z8lXo36VD)
</Card>

## Beneficios clave

- Simplifica integraciones con plataformas como Shopify, Stripe, Intercom, HubSpot y más.
- Funciona con **consultas en lenguaje natural**.
- Da acceso a herramientas externas directamente dentro de los prompts del modelo.
- Reduce el tiempo de configuración de horas a minutos.

## Dónde encontrarlo

En el menú izquierdo, haz clic en **AI Hub** y ahí verás los **Servidores MCP**.

## Cómo integrar un servidor MCP

- Puedes conectar **cualquier servidor MCP personalizado**, incluso auto-hospedado (self-hosted).
- Puedes conectar con OpenAI, Anthropic, etc.
- Sigue siempre el mismo proceso: **Nombre → URL → Autenticación (opcional) → Seleccionar herramientas → Guardar → Usar en flujos.**

### Ejemplo: conectar Shopify

#### Paso 1

Ve a **OpenAI Playground → Tools → Add MCP Server**. Desde ahí puedes integrar varias herramientas como Shopify, Stripe, Intercom, HubSpot y más, y copiar la URL generada para usarla en tu integración.

<Frame caption="OpenAI Playground → Tools → Add MCP Server">
  <Placeholder />
</Frame>

#### Dashboard de OpenAI

Ingresa la URL de tu tienda para generar la URL del servidor MCP, ver la autenticación requerida (si aplica) y la lista de herramientas disponibles.

<Frame caption="Dashboard de OpenAI con la URL del servidor MCP y las herramientas">
  <Placeholder />
</Frame>

#### Paso 2 — Agregar un servidor

Para agregar un servidor, simplemente ingresa el **nombre**, la **URL** y los datos de **autenticación** requeridos.

</Frame>

## Cómo usarlo dentro del Constructor de Flujos

- Ve a **Constructor de Flujos** → agrega un **bloque de Acción** → en **Integraciones**, selecciona **OpenAI**.
- Elige **"Create Model Response"** → cambia el modelo a **GPT-4.1-Mini**.

<Card type="warning">
  Se recomienda **GPT-4.1-Mini** por su mejor compatibilidad con herramientas MCP. **Evita GPT-4o-mini / 4o.**
</Card>

- Selecciona los **Servidores MCP**, configura los tokens y la temperatura.

<Frame caption="Configuración de Create Model Response con los Servidores MCP">
  <Placeholder />
</Frame>

### Datos de ejemplo

En la respuesta de ejemplo recibirás una lista de datos. Identifica y selecciona la **ruta correcta** (resaltada en el ejemplo), ya que es la que lleva a los resultados deseados. Extrae los datos de esa ruta y guárdalos en una variable.
