Las **Palabras clave en comentarios (Comment Keywords)** permiten que tu bot responda automáticamente cuando un usuario comenta en tus publicaciones de Facebook o Instagram. El bot puede dar *like* a los comentarios, responder públicamente y enviar mensajes privados a quienes comentan.

## Acceder a Comment Keywords

Ve a **Automatización → Palabras clave en comentarios** y haz clic en el botón **+ Comment Keyword** (esquina superior derecha).

<Frame caption="Sección de Comment Keywords y botón + Comment Keyword">
  <Placeholder />
</Frame>

## Pasos de configuración

### Paso 1: Selecciona la publicación

Elige la publicación de Facebook o Instagram que quieres monitorear. Si no la encuentras, puedes elegir **ID de publicación personalizado** e insertarlo.

<Frame caption="Selección de la publicación a monitorear">
  <Placeholder />
</Frame>

### Paso 2: Define las condiciones del disparador

Configura las palabras clave y el tipo de coincidencia. Opciones disponibles:

| Condición | Descripción |
|---|---|
| **Is** | Coincide exactamente con alguna palabra clave de la lista. |
| **Contains** | Se dispara si la palabra aparece en cualquier parte del comentario (por defecto). |
| **Starts with** | El comentario debe empezar con la palabra clave. |
| **Is anything** | Se dispara con cualquier comentario (el campo de palabra se desactiva). |

### Paso 5: Guardar payload del comentario (avanzado)

Activa esta opción para guardar los datos del comentario (ID de usuario, texto del comentario y otros detalles) en una **variable JSON**, para casos de uso avanzados.

Esto te permite usar la información del comentario dentro de tu subflujo de mensaje privado para crear respuestas personalizadas con IA. Por ejemplo:

- **Respuestas de DM personalizadas:** usa integraciones de IA dentro de tu flujo para generar mensajes privados personalizados según el contenido del comentario.
- **Respuestas públicas con IA:** genera respuestas públicas contextuales usando IA. Esto requiere usar el nodo de Acción avanzada de **Instagram API** o **Facebook API** dentro del subflujo (detallado en la sección de caso de uso más abajo).

# Usar IA para responder automáticamente los comentarios de tus Facebook Ads

## Introducción

Existe una función mediante la cual, cuando un usuario comenta en tu *dark post* o anuncio, puedes darle un **Like** y **responder** a esos comentarios bajo el disparador de Comment Keyword.

## Crear la publicación del anuncio (FB Ad Post)

Crea una publicación de anuncio en Ads Manager.

  <Placeholder />
</Frame>

## Crear el disparador de Comment Keyword

Ve a **Automatización → Comment Keywords → Crear nuevo Comment Keyword**.

<Frame caption="Creación de un nuevo Comment Keyword">
  <Placeholder />
</Frame>

Una vez que haces clic en Nuevo Comment Keyword, configuras esta función para que se dispare con los comentarios de tu FB Ad Post. Configura los parámetros requeridos:

<Frame caption="Parámetros del Comment Keyword para el anuncio">
  <Placeholder />
</Frame>

1. Selecciona el **FB Ads Post** para el cual necesitas responder los comentarios y dar likes.
2. Comentario en **is Anything** (para dispararse con cualquier tipo de comentario).
3. Público (Public) - **No se usa**.

Estado en **Activo**.

1. **Guardar payload del comentario** (la información del comentario junto con el ID de usuario y otros detalles se almacena en esta **variable JSON** para recuperar datos después).
2. **Respuesta privada en Messenger** (selecciona el **subflujo** que se ejecuta cuando ocurre el disparador del comentario).

Las siguientes opciones se activan (**Active**):

- Responder a nuevos usuarios
- Responder a suscriptores existentes
- Dar like a los comentarios
- Responder a las respuestas de los comentarios

## Crear el flujo del disparador de comentarios

Ve al subflujo, que llamamos **Facebook ads comment trigger**. El concepto para construir este subflujo:

1. Crea un **bloque de Condición** para diferenciar entre el comentario con la palabra clave **chatgpt** y las preguntas generales sobre tu negocio o servicios.
2. Cuando el comentario es **chatgpt**, disparas un bloque de Acción donde llamas a la **Facebook API** con la función **Ads Post Comment Reply**.
3. En esa función de Acción, recuperas y mapeas el **Post ID o Comment ID** desde la variable JSON declarada en la automatización del disparador.
4. Luego envías un mensaje **PM** al usuario por su canal de Messenger.
5. Si el comentario **no** es **chatgpt**, diriges a otro bloque de Acción con integración de **OpenAI**, donde haces un **Embedding Match** con el comentario del usuario para comprobar si está relacionado con tu negocio o servicios, y recuperas el **openai_embed_text** relacionado.
6. Si el **Embed Score es > 0.8**, disparas un bloque de Acción con la integración **OpenAI - Create Chat Completion**.
7. Ahí creas una nueva variación del output mapeado de **openai_embed_text** y envías esa respuesta de OpenAI al comentario del usuario en el FB Ad Post.
8. Si el **Embed Score es < 0.8**, el comentario recogido en la variable JSON se envía al bloque de Acción con **OpenAI - Create Chat Completion**, con un nuevo mensaje de sistema para que OpenAI responda o salude según corresponda si es un mensaje de saludo.
