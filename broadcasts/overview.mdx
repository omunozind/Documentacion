Usa la función de **Difusiones (Broadcasts)** para enviar mensajes masivos a tus usuarios. Solo necesitas unos pocos ajustes.

Ve a **Difusiones** en la barra lateral izquierda y haz clic en **+ New Broadcast**.

## Configurar una difusión

<Steps>
  <Step title="Título de la difusión">
    Ponle un título (es para ti, no lo ven los usuarios).
  </Step>
  <Step title="Selecciona un subflujo">
    Elige el subflujo que se enviará.
  </Step>
  <Step title="Agrega condiciones">
    Agrega condiciones para filtrar quién recibirá este mensaje de difusión.
  </Step>
</Steps>

<Frame caption="Configuración de título, subflujo y condiciones">
  <Placeholder />
</Frame>

<Card type="note">
  **Tip:** para enviar a **todos** los usuarios, prueba la condición **"If User Id has any value"** (si el User Id tiene cualquier valor).
</Card>

  <Placeholder />
</Frame>

Completa el resto de los ajustes, como el **tipo de mensaje** y la **hora de envío**, y haz clic en **Save**.

<Card type="note">
  Si eliges **"Now"** en la programación, el mensaje se envía apenas haces clic en **Save**. Si no, puedes cancelar la difusión en cualquier momento antes de que se envíe.
</Card>

## Máximo de usuarios por minuto (Max Users Per Minute)

El ajuste **Max Users Per Minute** te permite controlar cuántos usuarios reciben el mensaje de difusión por minuto. Esto ayuda a enviar los mensajes de forma más gradual, haciendo que la difusión parezca más humana y menos automatizada.

<Card type="note">
  Video de referencia: [youtube.com/watch?v=PnJrnMWdLQw](https://www.youtube.com/watch?v=PnJrnMWdLQw)
</Card>
