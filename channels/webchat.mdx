El canal de **Chat web** te permite agregar un chatbot a cualquier sitio web sin depender de otro canal. Solo está disponible dentro de un **Omnicanal**.

<Card type="note">
  Toda la configuración del widget (contenido, ajustes, apariencia, estilos de visualización, Custom User ID, CSS y personalización avanzada) está documentada en detalle en la página **[Widget de Chat web](/webchat-widget/overview)**.
</Card>
