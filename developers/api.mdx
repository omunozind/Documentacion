## Introducción a las APIs

La **API de Botky** se usa para obtener datos del flujo que construyes. Estos son algunos ejemplos de lo que puedes hacer con la API de Botky:

- Obtener el flujo que has construido en Botky.
- Crear, eliminar u obtener datos, incluyendo el tag de flujo o el campo de flujo (flow field).
- Trabajar con los datos de los suscriptores: obtener su información, editar su tag o campo, así como el subflujo enviado a los suscriptores.
- Configurar todos los datos usados en ecommerce, como carritos o productos.

Si necesitas trabajar con los datos anteriores, las siguientes secciones te explican los pasos para usar la API de Botky.
