---
title: "Almacén de datos (Data Store)"
description: "Conecta Botky directamente a tu propia base de datos externa"
icon: "database"
---

Botky se integra de forma nativa con las siguientes bases de datos externas:

<CardGroup cols={3}>
  <Card title="MySQL" icon="database" />
  <Card title="MariaDB" icon="database" />
  <Card title="PostgreSQL" icon="database" />
</CardGroup>

<Card type="note">
  Cada una de estas conexiones tiene su propia configuración específica (credenciales de acceso, host, puerto, etc.) dentro de la sección de Integraciones.
</Card>

Una vez conectada tu base de datos, puedes usarla desde el nodo de Acción para leer o escribir información directamente en tus propias tablas, sin depender únicamente de los campos personalizados de Botky.
