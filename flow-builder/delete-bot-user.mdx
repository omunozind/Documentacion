Te permite definir un límite de usuarios para tu chatbot y eliminar automáticamente el exceso, simplificando la gestión de tu base de usuarios.

## Opción 1: nodo de Acción

Encuentra el paso **Eliminar usuario del bot** dentro de **Acciones básicas**. Cuando el flujo llega a este paso, el usuario se elimina automáticamente, y no se procesan más mensajes ni condiciones para él.

## Opción 2: límite de usuarios del bot

En **Configuración → General**, busca la opción **Límite de usuarios del bot**. Al definir un valor mayor a cero, el sistema revisa periódicamente el total de usuarios y elimina a los más antiguos si se supera el límite.

## Monitorear la cantidad de usuarios

Ve a la sección de **Analítica** para ver el total de usuarios de tu bot — esto te ayuda a decidir un límite adecuado.

## Otras opciones

Para una gestión más personalizada, puedes crear disparadores (triggers) basados en condiciones específicas — por ejemplo, eliminar a un usuario si no ha interactuado durante un período determinado. Los disparadores basados en tiempo te dan más control sobre el proceso de eliminación.
