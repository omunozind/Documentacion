Find quick answers to the most common questions about using Botky — from setting up your Partner API token and funding your wallet, to managing client workspaces, configuring custom pricing, and understanding how add-ons and Stripe top-ups work.

<AccordionGroup>

  <Accordion title="What is Botky?">
    Botky is a WordPress plugin built for UChat reseller partners. It gives you a branded dashboard to create and manage client workspaces, control plans and add-ons, handle billing through a built-in wallet, and top up your balance via Stripe — all from a single interface without leaving WordPress.
  </Accordion>

  <Accordion title="Do I need a UChat partner account?">
    Yes. Botky connects directly to the UChat platform on your behalf. You must have an active UChat reseller partner account and a valid **Partner API Token** from [uchat.com.au](https://uchat.com.au) before you can use any feature of the dashboard.
  </Accordion>

  <Accordion title="How does the wallet work?">
    The wallet holds a pre-funded USD balance that Botky draws from each time you create a workspace, change a plan, or activate an add-on. Before performing any of those operations, make sure your balance covers the full cost. You can add funds at any time through the **Top-ups** section using Stripe.
  </Accordion>

  <Accordion title="What happens if my wallet balance is insufficient?">
    The operation is immediately blocked and you see an **"Insufficient Balance"** error that shows the amount required alongside your current balance. Add the necessary funds via the Top-ups section and then retry the operation.
  </Accordion>

  <Accordion title="Can I set my own prices for plans and add-ons?">
    Yes. Your WordPress administrator can configure custom pricing directly in your WordPress user profile settings. Prices are stored per WordPress user, which means different administrators on the same site can each maintain their own independent pricing for plans and add-ons.
  </Accordion>

  <Accordion title="What languages does Botky support?">
    Botky supports **English** and **Spanish**. The language is detected automatically from your WordPress site locale — sites whose locale code starts with `en` display in English, while all other locales default to Spanish. You can change the site language in **WordPress Admin → Settings → General → Site Language**.
  </Accordion>

  <Accordion title="Can clients access the dashboard without WordPress admin access?">
    Yes. Add the `[botky_panel]` shortcode to any WordPress page and clients can reach the full dashboard by logging in with their standard WordPress credentials. They do not need administrator rights — the shortcode renders the entire Botky interface for any authenticated user.
  </Accordion>

  <Accordion title="What happens when I delete a workspace?">
    Deletion is **permanent and irreversible**. All workspace data is removed from UChat immediately and cannot be recovered. Botky displays a confirmation prompt reminding you that this action cannot be undone and all data will be lost — confirm only when you are certain.
  </Accordion>

  <Accordion title="How are add-on costs deducted?">
    When you activate an add-on, the cost is deducted from your wallet immediately at the moment the request is sent. If the UChat API call subsequently fails, Botky automatically refunds the full amount back to your wallet so you are never charged for an unsuccessful operation.
  </Accordion>

  <Accordion title="Why is my connection status showing 'No API Key'?">
    You have not yet saved a Partner API Token for your account. Open the **Settings** tab in the Botky dashboard, paste your UChat Partner Token into the token field, and click **Save**. Once saved, the connection status indicator in the sidebar will update to confirm the token is active.
  </Accordion>

  <Accordion title="Where does payment go after a Stripe top-up?">
    After Stripe confirms your checkout session, it sends a webhook notification directly to your WordPress site. Botky listens for that webhook and automatically credits the payment amount to your wallet balance. The entire process typically completes within a few seconds of payment.
  </Accordion>

  <Accordion title="Can multiple WordPress users each have their own API token?">
    Yes. Every WordPress user maintains their own independent Partner API token and wallet balance. One user's token and funds have no effect on another user's account, making it straightforward to operate multiple reseller accounts or administrator roles on a single WordPress installation.
  </Accordion>

</AccordionGroup>
