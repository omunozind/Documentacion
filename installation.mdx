Installing Botky takes only a few minutes and requires nothing beyond standard WordPress admin access. Once activated, the plugin adds a **Partner Connect** menu to your WordPress admin sidebar and optionally exposes a front-end dashboard via shortcode — so you can choose whether partners manage workspaces from the WordPress admin or from a branded page on your site.

## Requirements

Before you install, confirm you have the following:

- **WordPress site with admin access** — you need the `manage_options` capability to install plugins and access the admin dashboard.
- **UChat Partner account** — Botky communicates with the UChat Partner API. You must have an active partner account at [uchat.com.au](https://uchat.com.au) and a valid Partner API token.
- **Stripe account** — the built-in wallet top-up flow uses Stripe for payment processing. You need a Stripe account configured by your Botky provider before you can accept top-up payments.

## Installation Steps

<Steps>
  <Step title="Download the Plugin ZIP">
    Obtain the `botky.zip` file from your Botky provider. This is the installable WordPress plugin package — do not unzip it before uploading.
  </Step>

  <Step title="Open the Plugin Upload Screen">
    Log in to your WordPress admin panel. In the left sidebar, navigate to **Plugins → Add New**, then click the **Upload Plugin** button at the top of the page.
  </Step>

  <Step title="Upload the ZIP File">
    Click **Choose File**, select the `botky.zip` file from your computer, and click **Install Now**. WordPress uploads and unpacks the plugin automatically.
  </Step>

  <Step title="Activate the Plugin">
    Once the upload completes, click **Activate Plugin**. WordPress activates Botky and registers all of its menus, shortcodes, and AJAX handlers.
  </Step>

  <Step title="Confirm the Menu Appears">
    Look at your WordPress admin sidebar. A new **Partner Connect** menu item should now be visible. Click it to open the Botky partner dashboard and confirm the installation was successful.
  </Step>
</Steps>

## Accessing the Dashboard

Botky gives you two ways to surface the partner dashboard, depending on your workflow.

### Admin Panel

Navigate to **Partner Connect** in the WordPress admin sidebar. This view is protected by the `manage_options` capability, so only WordPress administrators can access it. Use this approach when you — the reseller — are the only person who needs to manage workspaces.

### Frontend Shortcode

Embed the shortcode below in any WordPress page or post to display the full partner dashboard on the front end of your site:

```plaintext
[botky_panel]
```

This approach lets you create a branded, white-labeled portal page that partners or sub-resellers can visit without ever entering the WordPress admin.

<Tip>
  The shortcode is ideal for giving partners their own branded front-end portal. Create a dedicated WordPress page (for example, `/partner-dashboard`), add the `[botky_panel]` shortcode, and share that URL with your partners. They get the full dashboard experience without needing any WordPress admin access.
</Tip>

<Warning>
  Users must be logged in to a WordPress account to access the front-end panel. Guests who visit a page containing `[botky_panel]` without being logged in will see a login prompt instead of the dashboard. Make sure your partners have WordPress user accounts on your site before you share the portal URL with them.
</Warning>
